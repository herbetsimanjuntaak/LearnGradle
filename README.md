# LearnGradle

**project description**
simple custom task accepts CLI parameters and prints them with a greeting message

**add depedency**
implementation 'com.google.guava:guava:29.0-jre'
testImplementation 'junit:junit:4.13'

**how to run** 
./gradlew greetingTask -Pnama=Herbet
